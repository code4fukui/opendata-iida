# opendata-iida

[
![デプロイ](https://github.com/code4fukui/opendata-iida/actions/workflows/pages/pages-build-deployment/badge.svg)
](https://code4fukui.github.io/opendata-iida/)

長野県飯田市のオープンデータ（文化財、子育て施設）を地図上に可視化するプロジェクトです。

## デモ (Demo)

- **[統合マップ (Combined Map)](https://code4fukui.github.io/opendata-iida/)**: 文化財（赤）と子育て施設（緑）を1つの地図にまとめて表示します。
- **[個別マップ (Individual Maps)](https://code4fukui.github.io/opendata-iida/map.html)**: 各データセットを個別の地図で表示します。
